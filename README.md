# AI Agents Project

Проект для изучения и разработки AI агентов с использованием Python.

## Требования

- Python 3.12+
- uv для управления зависимостями

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd agents-for-david
```

2. Установите зависимости с помощью uv:
```bash
uv sync
```

3. Активируйте виртуальное окружение:
```bash
uv shell
```

## Запуск

```bash
uv run python main.py
```

## Структура проекта

- `main.py` - основной файл приложения
- `pyproject.toml` - конфигурация проекта и зависимости
- `.python-version` - закрепленная версия Python
- `uv.lock` - файл блокировки зависимостей

## Разработка

Проект использует uv для управления зависимостями и виртуальным окружением.
Для добавления новых зависимостей используйте:

```bash
uv add package-name
```

Для разработческих зависимостей:

```bash
uv add --dev package-name
```
